# Super Quick Setup Guide

1. Download and install Ollama from [ollama.ai](https://ollama.ai)

2. Open a terminal/command prompt and run these commands:
   ```bash
   # Pull the CodeLlama model to your machine (one-time download, about 4GB)
   ollama pull codellama:7b
   
   # Start the Ollama server
   ollama serve
   ```

3. Install the AIQuickFix extension in VS Code

4. Start coding! When you have errors, click the lightbulb to see AI-suggested fixes

That's it! Everything runs locally on your machine - no API keys, no cloud services.
